## 🚀 Installation

### Step 1 — Download

👉 Go to [**Releases**](https://github.com/sgsaifyt87/Instagram-Video-Download-For-Pc/releases/latest)
and download **`ZenithInstaDL_Setup.exe`**

### Step 2 — Install

- Double-click `ZenithInstaDL_Setup.exe`
- Click **Next** → **Install**
- Done ✅

### Step 3 — Run

- A shortcut appears on your **Desktop**
- Double-click **Zenith InstaDL** and start downloading!
